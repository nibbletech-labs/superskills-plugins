# Subagents

This directory contains custom subagents for specialized tasks.

## Structure

Each subagent should have its own folder:
```
agents/
├── my-agent/
│   ├── AGENT.md          # Agent definition
│   └── README.md         # Documentation
└── another-agent/
    ├── AGENT.md
    └── README.md
```

## Usage

Subagents can be:
1. Installed standalone by users
2. Bundled into plugins
3. Distributed via the SuperSkills marketplace

## Examples

- Specialized research agents
- Code analysis agents
- Documentation generators
- Testing assistants
